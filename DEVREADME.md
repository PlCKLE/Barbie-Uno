HOW TO RUN NEW BARBIE UNO

Step 0: If you don't have it already, install node.js. https://nodejs.org/en

Step 1: In the terminal, run this command: cd barbie-uno

Step 2:In the terminal, run this command: npm install

Step 3: In the terminal, run this command: npm run dev

Step 4: You're officially running the server!

Step 5: Click the link.
